# Install Blender for ARIA-OS Lattice Generation

ARIA-OS uses Blender headless for lattice generation.
Blender is free and open source.

## Install
1. Go to https://www.blender.org/download/
2. Download Blender 4.x for Windows
3. Install to default location
4. Run: blender --version to verify

## After Install
Run: python run_aria_os.py --lattice-test
to verify the pipeline works.
